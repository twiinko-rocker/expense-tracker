# expense-tracker

A full-stack CRUD application to manage personal expenses.
It includes user authentication, a REST API, and a React frontend — covering the core patterns used in real-world apps.

Tech Stack

* Frontend: React
* Backend: Node.js + Express
* Database: MongoDB (Mongoose)
* Auth: JWT

Features (MVP)

* User registration & login
* Add, edit, delete expenses
* View all expenses
* Simple dashboard with total balance

Run Locally

1. Clone the repo

git clone <your-repo-url>
cd <your-project-folder>

2. Server

cd server
npm install
npm run dev

3. Client

cd client
npm install
npm run dev
